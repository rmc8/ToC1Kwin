プロセカ楽曲効率

プロセカ楽曲の効率表です。Expertで出せるスコアを100%としたときに同じ楽曲で異なる難易度のスコア効率が良いか悪いかを表に表します。低難易度でもスコアが落ちにくいもの・高難易度でスコアが伸びやすいものが「Y」、スコアが伸びにくい楽曲は「N」、標準的な効率であれば記載なしの書式で表に効率を掲載しております。

低難易度はスキルの影響を強く受けやすいため、スキルずれが起こりづらくスキル効果を巻き込みやすい1枠で活用することを推奨します。またMasterはだいたいの曲はAPをだしておけばスコアがExpertわずかに伸びる曲も多いので「N」判定は厳しめにつけています。1000連勝以上を狙うなど長時間ハイスコアをだし続けるための周回では高難易度の回避にご活用ください。

また一部でMasterやExpertよりもHard以下がスコアが出やすい曲があったり、高難易度と同等のスコアが出る曲があったりします。しかしながら、基本的にはチアフルイベランでライフを残しつつイベントポイント（スコア）を稼ぐ用途で使うことが望ましく、連勝や討伐で厳しくスコアを求める場面の場合には表をうのみにするだけではなく自分なりに目的を持って検証して、正しい理解の下で難易度の判断ができるようにお願いいたします。
